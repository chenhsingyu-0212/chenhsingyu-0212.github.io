官方文件
https://containrrr.github.io/watchtower/

# 介紹

WatchTower 啓動後，會開始幫你監控機器上所有正在執行的 container，每隔一段時間發現有新的 image 後，它就會幫你下載最新 image 並且使用你當初建立該 container 時的相同環境設定去幫你重新啓動容器。

# docker 開啟 WatchTower
