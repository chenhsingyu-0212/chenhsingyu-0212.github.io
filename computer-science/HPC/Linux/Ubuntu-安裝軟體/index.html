# 安裝 anaconda

前往 anaconda 網站進行下載

下載後會得到一個 ``

  1. 利用 bash [下載檔案名稱.sh] 開始安裝，接著按下 Enter 會有一段授權條款，可以用上下鍵看內容
  2. 接著會問你是否同意，輸入 yes
  3. 接著確認安裝位置，按下 Enter
  4. 接著安裝好後，記得重新開機
  5. 電腦重開後，終端機提示命令符會看到 (base) 在前面
  6. 可以利用 conda deactivate 退出 conda 環境
