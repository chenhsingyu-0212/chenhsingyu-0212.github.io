# What is tmux?

tmux 是一個終端機管理工具，可以分割視窗、同時開啟多個終端機。

  • 如果想要同時使用多個命令，或執行多個任務時，就會很方便
  • 如果是遠端 ssh 連線到其他主機使用 tmux 來執行程式，也可以避免 ssh 突然斷連後，正在安裝或執行的任務被終止
