# 設定 git 識別資料

在你安裝 Git 後首先應該做的事是設定使用者名稱及電子郵件。
這一點非常重要，因為每次 Git 的提交會使用這些資訊，而且提交後不能再被修改。

若你有傳遞 --global 參數，只需要做這工作一次，因為在此系統，不論 Git 做任何事都會採用此資訊。
