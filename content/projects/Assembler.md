> University coursework — an assembler built from scratch.

**Assembler** is a **Java** implementation of an assembler that translates assembly-language
source into machine code, producing an operation-code table and the assembled program
output. Built as a systems-programming course assignment.

## Highlights

- Parses an assembly input file and emits the **op-code** and **assembled-program** outputs.
- Written primarily in **Java** (with Python / C++ helpers), runnable via a batch script.
