## 製作開機機碟

可以看官方提供的文件

- [Create a bootable USB stick with Rufus on Windows](https://ubuntu.com/tutorials/create-a-usb-stick-on-windows#1-overview)

### 需要的東西

- 一個 4GB 以上的隨身碟(空的或格式化過的)
- Windows 作業系統
- Rufus 軟體, a free and open source USB stick writing tool
- Ubuntu ISO file

1. 到 ubuntu 的官網下載 ISO 檔
   - [Ubuntu 22.04.4 LTS](https://releases.ubuntu.com/jammy/)
   ![Imgur](https://i.imgur.com/0KtyYUs.png)
2. 安裝 [Rufus](https://rufus.ie/zh_TW/)
3. 插入 USB 隨身碟
4. Rufus 將更新，在設備欄位中設定設備

<img src="https://i.imgur.com/g18T5oJ.png" alt="Rufus" width="350">

## 安裝單一系統

1. 製作開機碟 / 拿到一個開機碟
2. 隨身碟插上去電腦，並重新開機
3. 開機後，狂按 F2 或是 delete 進入 BIOS
4. 在右下角的開機硬碟處把隨身碟拉到最上面，接著儲存後重開機

<img src="https://i.imgur.com/HccUxFF.png" alt="BIOS" width="600">

<img src="https://i.imgur.com/1USVcrs.png" alt="BIOS" width="600">

5. 安裝 Ubuntu，選擇清除所有硬碟安裝

## 安裝雙系統

1. 製作開機碟 / 拿到一個開機碟
2. 磁碟切割，配置 Ubuntu 的安裝空間，如果是插新的硬碟或 SSD 就不用
   - 關閉 Windows 快速啟動
3. 關機重開，進入 BIOS
4. 將 boot option 順序調整為 ubuntu 在前面(UEFI: USB ......)
5. 按下 F10 後儲存 BIOS 設定，重開電腦
6. 接著就會進入 Ubuntu 開機選單

## Ubuntu 設定

1. 選擇 install ubuntu
2. 選擇鍵盤語言
3. 選擇安裝選項
   - 可以選擇**一般安裝**或**最小安裝**
   - 其他安裝選項**在安裝時同時下載更新**或**安裝適用於圖形和 Wi-Fi 硬體以及其他媒體格式的第三方軟體**
4. 選擇要如何安裝
   - 與 Windows 啟動管理器一起安裝 ubuntu
   - 清除磁碟並安裝 ubuntu

<img src="https://i.imgur.com/iflnAIn.png" alt="Imgur" width="600">

5. 選擇時區
6. 設定使用者名稱、密碼等資訊
7. 接著就開始安裝啦~~

## 設定網路

1. 查詢網路介面名稱

   ```shell
   # Show all IP addresses associated with all network devices.
   ip a
   # Display link layer information, including characteristics of link layer devices currently available.
   ip link
   ```

2. 編輯設定檔，路徑 `/etc/netplan`，有一個檔案 `network-manager.yaml`
3. 加入網路資訊

   ```yaml 固定 IP
   network:
      version: 2
      renderer: NetworkManager
      ethernets:
         enp5s0:
         addresses: [120.108.204.64/24]
         gateway4: 120.108.204.254
         dhcp4: false
         nameservers: 
            addresses:
               - "8.8.8.8"
               - "1.1.1.1"
   ```

4. 啟動網路設定

   ```shell
   # Use this to try the network is set OK, it will recover originally with 120 seconds.
   sudo netplan try
   # Start new network settings.
   sudo netplan apply
   ```
