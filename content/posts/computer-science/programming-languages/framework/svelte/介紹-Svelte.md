# 簡介
這是一個在 2019 年，衝進大家視眼裡的一個前端框架。

在介紹此框架前，大家可能比較熟悉的三大前端框架是: React、Vue 或 Angular。
而這個前端框架有別於這三種，留一些懸念看下去吧!

# 特色
當應用程式運行時，像 React 和 Vue 等框架會在使用者的瀏覽器直接做它們想要做的事情，而 Svelte 則是將做的這些事移到編譯階段處理，因此那些做的事情只會發生在你建置應用程式的時候，進而能產生高度優化的原生 JavaScript 程式碼。
