# 基本介紹

  • 出處：左傳
  • 背景：秦、晉合兵圍攻鄭，佚之狐向鄭文公推薦燭之武說服秦穆公撤兵。
  • 出場人物:
    • 佚之狐 - 善於視人
    • 燭之武 - 有勇有謀、能言善辯

# 作者介紹

左丘明，生卒年不詳，為春秋時魯國太史，可能與孔子同時期。

# 內容

[經] - (春秋原文)
晉人、秦人圍鄭。

[傳]
九月甲午，晉侯、秦伯圍鄭，以其無理於晉，且貳於楚也。晉軍函陵，秦軍氾南。

佚之狐言於鄭伯曰:「國危矣！若使燭之武見秦君，師必退。」公從之。辭曰:「臣之狀也，猶不如人，今老矣！無能為也已。」公曰:「吾不能早用子，今急而求子，是寡人之過也。然鄭亡，子亦有不利焉！」許之。夜縋而出。

見秦伯曰:「秦、晉圍鄭，鄭既知亡矣！」
